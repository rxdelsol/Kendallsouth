Kendall South Credentialing - Full project (navy theme)

Instructions:
1. unzip and run `npm install`
2. run `npm run dev` to start local dev server
3. The app stores data in localStorage keys: doctorsList, doctorsByInsurance, insuranceList
4. Use the Doctors table to add providers (NPI, license, CAQH, Medicaid, Medicare, DOB, taxonomy)
5. Use Insurances modal to assign doctors to insurances and set expiration + notes
6. Export/Backup/Restore/CSV available in action bar
